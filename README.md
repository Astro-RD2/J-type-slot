# REAMDE.md for project J-type-slot

## 專案概述
- 永宙公司委託泰偉電子合作開發 J型老虎機  
  使用一實體 slot machine 的機構, 內部放置一 PC box 執行瀏覽器連線到遊戲網頁.  
  玩家紙鈔機入金, 然後使用觸控螢幕、實體鍵盤來進行遊戲，最後透過收據印表機印出結餘資訊，拿到櫃檯實際出金.  
  2025/4/2 正式立案. 預計開發時間 1 ~ 2 個月.
